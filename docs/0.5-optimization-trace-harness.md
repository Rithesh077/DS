# 0.5 optimization trace harness (scipy optimize)

i keep running into "it converged" without knowing what actually happened.
this is a tracer/visualizer for scipy optimizers and root finders.

## v1

- run a method on a known problem
- capture per-iteration info (or as close as the api allows)
- plot convergence curves and compare methods side-by-side

## baselines

- root finding: bisect, brentq, newton
- minimization: a few minimize methods (start with nelder-mead, bfgs, powell)

## test problems

- quadratic (sanity check)
- rosenbrock
- ill-scaled bowl
- noisy objective (where "convergence" is ambiguous)

## outputs

- termination message + status code
- number of evaluations / iterations
- best x found + f(x)
- trace plots (f vs iteration; step size when possible)

## upstream notes

- cases where messages/status codes are misleading
- methods that fail silently or stop too early
