# 0.6 linear algebra stability explorer (scipy linalg / sparse)

small experiments to build intuition for conditioning and solver choice.
main point: show how tiny perturbations in a or b can blow up the solution.

## v1

- ill-conditioned matrices (hilbert, vandermonde) and how conditioning scales with n
- scaling experiments (same problem, different scaling -> different numerical behavior)
- dense vs sparse solvers on comparable systems
- sensitivity plots: ||dx|| vs ||da|| / ||db||

## what to measure

- residual norm vs true error (when ground truth is known)
- condition number and rank estimates
- time/memory for dense vs sparse

## upstream notes

- solver defaults that surprised me
- sparse solver failure modes that deserve clearer documentation
