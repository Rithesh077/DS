# 0.8 autograd and numerical stability lab (pytorch)

small experiments for when models blow up: custom autograd, gradient checking, and stable implementations of common ops.

## v1

- implement 1-2 custom torch.autograd.function and verify gradients
- use torch.autograd.gradcheck + finite differences
- reproduce stability tricks:
  - log-sum-exp
  - stable softmax / cross-entropy
  - stable variance / standard deviation
- compare optimizers on controlled objectives and note failure modes

## failure modes to trigger

- nans/infs from overflow in exp/log
- exploding gradients from bad scaling
- non-smooth points where gradcheck fails (and why)
