# 0.2 distribution fitting engine (scipy)

small fitter/ranker: given 1d data, fit candidate distributions, compare them, and plot the winners.

## v1

- input: csv vector (plus a seeded synthetic generator)
- candidates: normal, exponential, gamma, lognormal
- per candidate:
  - log-likelihood
  - aic
  - a simple gof check (start with ks; see caveat)
- outputs:
  - ranked table (csv/json)
  - histogram + pdf overlays for the top 2-3

## gotchas

- support matters (do not fit positive-only distributions to negative data)
- parameter order is easy to mix up (shape(s), loc, scale)
- fit() can converge to weird params; log them and surface warnings
- ks p-values are optimistic when params are estimated on the same data
  - if ks matters here: use a parametric bootstrap p-value or a held-out split

## upstream notes
