# 0.1 regression diagnostics engine (statsmodels)

small cli to answer: is this linear regression model defensible for this dataset?

## v1

- input: csv (plus a seeded synthetic generator for quick testing)
- fit: ols via statsmodels
- outputs:
  - a short markdown report (key coef + fit stats + warnings)
  - a diagnostics table (test, statistic, p-value, note)
  - plots saved under outputs/figures/ (residuals vs fitted, qq, leverage/cook)

## checks (v1)

- normality: shapiro-wilk + qq
- heteroscedasticity: breusch-pagan
- multicollinearity: vif
- autocorrelation: durbin-watson (add ljung-box when the data is ordered)

## edge cases to reproduce

- singular / near-singular design matrices
- missing values and constant columns
- small n (tests get noisy; p-values stop being helpful)
- outliers + high leverage points (where "significance" flips)

## upstream notes
