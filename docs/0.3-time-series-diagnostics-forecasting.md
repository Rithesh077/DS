# 0.3 time series diagnostics and forecasting (statsmodels.tsa)

repeatable checklist for a single time series: stationarity, baseline fit, and residual sanity.

## v1

- input: univariate series with a timestamp column
- baseline: arima / sarimax
- report/plots:
  - plot + missingness summary
  - stationarity checks (adf + kpss) with a plain-language note
  - acf/pacf plots
  - fit summary (params + aic)
  - residual diagnostics (ljung-box + a quick normality check)
  - forecast + prediction intervals

## gotchas

- irregular timestamps and duplicate dates
- strong seasonality hiding as "autocorrelation"
- stationary in mean but not in variance
- structural breaks (fits early, fails late)
